Next activity

This setting specifies a Moodle activity to be done after this Quizport is completed. The optional grade is the minimum grade for this QuizPort that is required for the next activity to be shown.

The teacher can select a specific activity, or a one of the following general purpose settings:

  • Next activity in this course
  • Next activity in this section
  • Next graded activity in this course
  • Next graded activity in this section
  • Next QuizPort activity in this course
  • Next QuizPort activity in this section

If other exit page options are disabled and the student has achieved the required grade on this QuizPort, the next activity will be shown straight away. Otherwise, the student will be shown a link to the next activity, which they can click when they are ready.
