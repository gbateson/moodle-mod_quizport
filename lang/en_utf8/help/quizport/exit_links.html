Exit page links

These options enable and disable the display of certain navigation links on a QuizPort's exit page.

Retry
If multiple attempts are allowed at this QuizPort and the student still has some attempts left, a link to allow the student to retry the QuizPort will be displayed

Index
If checked, a link to the QuizPort index page will be displayed.

Course
If checked, a link to the Moodle course page will be displayed.

Grades
If checked, a link to the Moodle gradebook will be displayed.
