Title

This setting specifies the title to be displayed on the web page.

Quiz name
The quiz name will be displayed as the web page title.

Get from source file
The title, if any, defined in the source file will be used as the web page title.

Use source file name
The source file name, excluding any folder names, will be used as the web page title.

Use source file path
The source file path, including any folder names, will be used as the web page title.

Additionally, if the Prepend unit name box is checked, the name of the unit will be added at the beginning of the title.

Similarly, if the Append sort order box is checked, the sort order of this quiz within this unit will be appended to the title.
