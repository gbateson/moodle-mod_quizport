Unit attempt grade method

This setting defines how the grade for a single unit attempt is calculated.

Total
The unit attempt grade will be set to the sum of the weighted quiz scores.

Highest
The unit attempt grade will be set to highest weighted quiz score.

Last
The unit attempt grade will be set to the weighted score of the most recently attempted quiz.

Last (completed)
The unit attempt grade will be set to the weighted score of the most recently attempted quiz whose status is "completed". Other attempts will be ignored.

Last (timed out)
The unit attempt grade will be set to the weighted score of the most recently attempted quiz whose status is "completed" or "timed out". Other attempts will be ignored.

Last (abandoned)
The unit attempt grade will be set to the weighted score of the most recently attempted quiz whose status is "completed" or "timed out" or "abandoned". Attempts whose status is "in progress" will be ignored.
