Quiz names

The quiz names can be derived from specfic text entered by the teacher or they can be automatically generated.

Get from source file
The quiz names will be extracted from the source files.

Use source file name
The source file names will be used as the quiz names.

Use source file path
The source file path of each quiz will be used as the name for that quiz. Any slashes in the file path will be replaced by spaces.

Specific text
A unique name for each quiz will be created by appending the quiz sort number to the specific text entered by the teacher.
